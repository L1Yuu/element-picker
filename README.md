## A JavaScript library to help you point and click to get the hovered element.

### Install

```bash
npm install --save element-picker
```

### Usage

```js
var elementPicker = require('element-picker')

function onClick(element) {
  console.log(element)
}

elementPicker.init({ onClick })
```
